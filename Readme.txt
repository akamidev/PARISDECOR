🎯 Objectif

Créer une présence web claire et crédible pour PARIS DECOR (peinture, vitrerie, carrelage, parquet) afin de :

Générer des demandes de devis (CTA et formulaires).

Rassurer via des réalisations et avis clients.

Travailler le SEO local (Saint‑Gratien / Val‑d’Oise / Île‑de‑France).

Offrir un design propre, rapide et responsive.

🚧 Périmètre

Pages principales : index.html, services.html, projects.html, project-details.html, quote.html, contact.html, team.html, terms.html, privacy.html.

Contenus 100% en français, orientés second œuvre (préparation des supports, peintures A+, vitrerie, carrelage SDB, parquet flottant/collé, petites rénovations).

Conservation de la structure HTML d’origine pour compatibilité CSS/JS, mais textes réécrits et adaptés au métier.

🧱 Stack & structure

Front : HTML5, CSS, JavaScript (animations AOS, slider Swiper, lightbox).

Assets : images optimisées (WebP) dans assets/img/…, icônes via Bootstrap Icons.

Code :

CSS principal : assets/css/main.css

JS principal : assets/js/main.js

Vendors : assets/vendor/* (AOS, Swiper, GLightbox, etc.)

Formulaires (si usage en PHP) : forms/quote.php, forms/get-a-quote.php, forms/contact.php
